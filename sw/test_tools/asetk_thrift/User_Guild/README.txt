Usage of asetk_thrift

1. Install 
     Microsoft .NET Framework 4 (dotNetFx40_Full_setup)
     https://www.microsoft.com/en-US/download/details.aspx?id=17718

     bonjoursdksetup.exe
     http://flow.tymphany.com/redmine/dmsf/files/17982/download


2. Let PC and ASE-TK on the same network


3. Execute Script:

      "asetk_print_all.bat" 
                 => Print all ASE-TK devices

      "asetk_print_by_ip_addr" 
                 =>  Print specific ASE-TK device.
                 =>  Please modify IP_ADDR of "asetk_print_by_ip_addr.bat"

      "asetk_print_by_name.bat" 
                 =>  Print specific ASE-TK device.
                 =>  Please modify Name of "asetk_print_by_name.bat"
